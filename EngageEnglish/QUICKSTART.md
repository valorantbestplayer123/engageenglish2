# Quick Start Guide - EngageEnglish v4

## Installation

```bash
# Install dependencies
pip install -r requirements.txt

# Run the application
python src/main.py
```

## Building Windows Executable

```bash
# Install PyInstaller
pip install pyinstaller

# Build executable
pyinstaller EngageEnglish.spec

# Find executable in dist/EngageEnglish.exe
```

## Game Modes

### 1. Speed Challenge 🏃
- Fast-paced multiple choice
- Time limit per question
- Streak bonuses
- **Best for**: Testing automaticity

### 2. Vocabulary Mastery 📚
- Synonym matching
- Progressive levels
- Star ratings (1-3)
- **Best for**: Building vocabulary depth

### 3. Context Builder 💡
- Sentence completion
- Words in context
- Real-world usage
- **Best for**: Reading comprehension

### 4. Practice Lab 🧪
- No HP penalties
- Hint system
- Unlimited retries
- **Best for**: Learning without pressure

## Controls

- **Mouse**: Click to select buttons, cards, and options
- **Escape**: Not used (use on-screen buttons)
- **Alt+F4**: Exit application

## Progress

- Progress saves automatically
- Track all 4 modes independently
- View stats in main menu footer

## Tips

- Start with **Practice Lab** to learn
- Use **Speed Challenge** to build automaticity
- Play **Vocabulary Mastery** for depth
- Try **Context Builder** for application
- Play daily to maintain streak!
